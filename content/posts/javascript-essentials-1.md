## Giới thiệu

JavaScript là một trong những ngôn ngữ lập trình phổ biến nhất hiện nay. Trong bài viết này, chúng ta sẽ cùng tìm hiểu về JavaScript và các khái niệm cơ bản về lập trình máy tính.

## JavaScript là gì?

JavaScript là ngôn ngữ lập trình được sử dụng chủ yếu để tạo các trang web tương tác. Ban đầu được phát triển để chạy trên trình duyệt web, nhưng giờ đây JavaScript có thể chạy trên cả server (Node.js) và nhiều nền tảng khác.

### Đặc điểm của JavaScript

- **Ngôn ngữ thông dịch**: Code được thực thi trực tiếp mà không cần biên dịch
- **Dynamic typing**: Không cần khai báo kiểu dữ liệu
- **Đa nền tảng**: Chạy được trên nhiều môi trường khác nhau
- **Hướng đối tượng**: Hỗ trợ lập trình hướng đối tượng

## Lập trình Máy tính cơ bản

### Thuật toán

Thuật toán là một tập hợp các bước được sắp xếp theo thứ tự để giải quyết một vấn đề cụ thể.

```javascript
// Ví dụ thuật toán đơn giản: Tính tổng 2 số
function tinhTong(a, b) {
    return a + b;
}

console.log(tinhTong(5, 3)); // Output: 8
```

### Cấu trúc chương trình

Một chương trình JavaScript cơ bản bao gồm:
- Khai báo biến
- Các câu lệnh xử lý
- Hàm (functions)
- Cấu trúc điều khiển

## Môi trường phát triển

Để bắt đầu với JavaScript, bạn cần:

1. **Trình soạn thảo code** (VS Code, Sublime Text, Atom)
2. **Trình duyệt web** (Chrome, Firefox) - có Developer Tools
3. **Node.js** (tùy chọn) - để chạy JavaScript ngoài trình duyệt

## Hello World với JavaScript

```javascript
// In ra console
console.log("Hello World!");

// Hiển thị alert trên trình duyệt
alert("Hello World!");

// Thay đổi nội dung HTML
document.getElementById("demo").innerHTML = "Hello World!";
```
